# First Pairing

## Fluxo básico

1. Subir o OpenClaw
2. Gerar QR/pairing code do WhatsApp
3. Aprovar sender quando necessário
4. Validar número conectado
5. Testar DM e grupos

## Dicas

- Aprovar sender não troca o número conectado do gateway
- Use logs para descobrir JIDs de grupos
- Teste sempre após reinício/config change
