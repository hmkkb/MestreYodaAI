# 2026-04-01 - Lições da configuração do WhatsApp

## Pontos principais

- Aprovar sender não é o mesmo que trocar o número conectado
- Grupos devem ser cadastrados por JID
- `/mestre` não é menção nativa
- `messagePrefix` não atua como hard-gate
- arquivo de memória diário ausente pode quebrar o bootstrap
- config inválida pode causar restart loop/turnos terminados
