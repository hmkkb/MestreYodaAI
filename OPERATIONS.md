# Operations

## Rotinas do dia a dia

- Ver logs do OpenClaw
- Reiniciar o gateway
- Validar conexão do WhatsApp
- Liberar grupos por JID
- Verificar cron jobs

## Exemplos de operação

### Ver logs
`docker exec -it openclaw-gwzl-openclaw-1 openclaw logs --follow`

### Reiniciar gateway
Use o mecanismo padrão do OpenClaw ou reinicie o container conforme o ambiente.
