# Troubleshooting

## Problemas já observados

- Configuração inválida causando restart loop
- Confusão entre grupo allowlist e sender allowlist
- Necessidade de JID real do grupo
- Falha por arquivo diário de memória ausente
- Respostas inesperadas sem hard-gate real por `/mestre`
- Warnings de plugin stale no startup

## Abordagem recomendada

- Confirmar logs
- Validar schema antes de alterar config
- Testar em grupos específicos por JID
