# OpenClaw WhatsApp Bot

Bot de WhatsApp construído com OpenClaw, operando em servidor Ubuntu hospedado na Hostinger.

## Objetivo

Centralizar a operação do agente, automações, grupos allowlisted, rotinas de manutenção e documentação operacional do projeto.

## Escopo

- Bot de WhatsApp via OpenClaw
- Operação em Ubuntu/Hostinger
- Grupos permitidos por JID
- Automação operacional
- Documentação técnica e runbooks
